Etiquetas html

En una página HTML que contiene texto, imágenes, videos y comandos de Bootstrap, puedes utilizar varias etiquetas HTML para estructurar y presentar tu contenido de manera adecuada. A continuación, te menciono algunas etiquetas junto con su función:

Define el inicio y fin del documento HTML.

Contiene información meta y enlaces a recursos externos, como hojas de estilo CSS y scripts JavaScript.

Define el título de la página que se muestra en la pestaña del navegador.

Contiene el contenido principal de la página.

Se utilizan para los encabezados de diferentes niveles, donde h1> es el más grande y h6> es el más pequeño.

Define un párrafo de texto.

Se utiliza para insertar imágenes en la página. Requiere el atributo src que especifica la ruta de la imagen.

Permite insertar videos en la página. Puedes utilizar los atributos src para especificar la ruta del video y controls para mostrar controles de reproducción.

Sirve como un contenedor genérico para agrupar elementos y aplicar estilos o manipularlos con JavaScript.

Permite aplicar estilos o manipular partes específicas de texto dentro de un párrafo u otro elemento.

Define un enlace a otra página o recurso. Requiere el atributo href que especifica la dirección URL de destino.

Crea un botón en la página que se puede utilizar para interactuar con el usuario.

Se utilizan para crear listas desordenadas, donde ul> representa la lista y li> representa cada elemento de la lista.

Se utilizan para crear listas ordenadas, donde ol> representa la lista numerada y li> representa cada elemento de la lista.

Permite incrustar contenido externo, como un mapa de Google o un reproductor de video, dentro de la página.
